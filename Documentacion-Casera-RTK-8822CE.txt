## 1. VISIÓN GENERAL DEL CHIP

El RTL8822CE es un chipset de Realtek con configuración Tx/Rx 2T2R, que cumple con los estándares IEEE 802.11a, 802.11b, 802.11g, 802.11n y 802.11ac, con interfaz de bus PCI Express. Opera en rangos de frecuencia de 5.15–5.85 GHz (802.11a/ac) y 2.412–2.484 GHz (802.11bg).

RTL8822CE combina un WLAN MAC, un baseband WLAN de 2T2R y RF en un solo chip.

El RTL8822CE MAC soporta 802.11e para aplicaciones multimedia, 802.11i y WAPI para seguridad, y 802.11n/802.11ac para eficiencia mejorada del protocolo MAC.

Además, RTL8822CE proporciona STBC (Space-Time Block Code) de un stream espacial, Transmit Beamforming (TxBF) y Low Density Parity Check (LDPC) para extender el alcance de transmisión.

La interfaz cumple con PCI Express Base Specification Revision 2.1, y USB2.0 FS-mode para Bluetooth.

### Características principales:

| Parámetro | Valor |
|---|---|
| **Chipset** | Realtek RTL8822CE-CG |
| **Estándar WiFi** | 802.11a/b/g/n/ac (WiFi 5) |
| **Bluetooth** | BT 5.0 + BLE |
| **Configuración MIMO** | 2T2R (MU-MIMO) |
| **Ancho de banda** | 20/40/80 MHz |
| **Tasa máxima (5 GHz)** | 867 Mbps |
| **Tasa máxima (2.4 GHz)** | 300 Mbps |
| **Interfaz WLAN** | PCIe 2.1 (1 lane) |
| **Interfaz BT** | USB 2.0 FS |
| **Form Factor** | M.2 2230 (Key A/E) |
| **Tensión** | 3.3V / 1.5A |
| **Temp. operación** | 0°C a +70°C |
| **MTBF** | > 150,000 horas |

### Tasas de datos soportadas:


- **802.11a:** 6, 9, 12, 18, 24, 36, 48, 54 Mbps
- **802.11b:** 1, 2, 5.5, 11 Mbps
- **802.11g:** 6, 9, 12, 18, 24, 36, 48, 54 Mbps


- **802.11n (HT):** Hasta 300 Mbps (2x2, 40 MHz, Short GI)
- **802.11ac (VHT):** Hasta 867 Mbps (2x2, 80 MHz, Short GI)

## 2. IDENTIFICACIÓN PCI / HARDWARE IDs

El PCI ID del RTL8822CE es `10ec:c822`. El Vendor ID es `10ec` (Realtek Semiconductor Co., Ltd.) y el Device ID es `c822`.

También existe una variante con Device ID `c82f` que igualmente corresponde al RTL8822CE.

### Tabla de Hardware IDs

| Campo | Valor |
|---|---|
| **PCI Vendor ID** | `0x10EC` |
| **PCI Device ID** | `0xC822` (principal) / `0xC82F` (variante) |
| **PCI Class Code** | `0x0280` (Network Controller: Wireless) |
| **PCI Revision** | `0x00` |
| **USB Vendor ID (BT)** | `0x0BDA` |
| **USB Device ID (BT)** | `0xC822` |
| **FCC ID** | TX2-RTL8822CE |
| **IC** | 6317A-RTL8822CE |

### Subsystem IDs conocidos:

Subsistemas conocidos incluyen AzureWave `1a3b:3750` (ASUS TUF), Hewlett-Packard `103c:85f7` (HP Stream), y Lenovo `17aa:c123` (ThinkBook 14).


### Subsistemas:

1. **SYS (System):** Control de power, clock, reset, EFUSE, reguladores
2. **MAC:** 802.11 MAC engine, TX/RX queues, encryption/decryption
3. **BB (Baseband):** OFDM/CCK/DSSS modulation, AGC, channel estimation
4. **RF:** Transceiver dual-band, synthesizer, PA, LNA
5. **BT:** Bluetooth 5.0 core con HCI sobre USB
6. **PCIe:** DMA engine, interrupt controller, bus mastering

## 4. INTERFAZ DE BUS PCIe

La interfaz PCIe incluye señales de PCI Express Transceiver Interface, Reference Clock Request Signal (también usado por L1 PM substates), y señal para solicitar el bus PCI Express.

La señal PCI Express Reset (PERST#) es activa baja. Cuando PERST# se activa en el estado de power-on, el RTL8822CE retorna a un estado de reset predefinido y queda listo para inicialización y configuración.

### 4.3 BAR (Base Address Registers)

Los recursos de hardware del RTL8822CE típicamente incluyen puertos I/O (256 bytes) y memoria MMIO de 64KB. Por ejemplo, un dispositivo puede reportar: "I/O ports at 2000 [size=256]" y "Memory at c0700000 (64-bit, non-prefetchable) [size=64K]".

```c
/* BAR Layout */
#define RTW_BAR0_IO_SIZE        256      /* I/O port space */
#define RTW_BAR1_MMIO_SIZE      0x10000  /* 64 KB Memory-Mapped I/O */

/* Acceso a registros */
static inline u8 rtw_read8(struct rtw_dev *rtwdev, u32 addr)
{
    return readb(rtwdev->mmio_base + addr);
}

static inline u16 rtw_read16(struct rtw_dev *rtwdev, u32 addr)
{
    return readw(rtwdev->mmio_base + addr);
}

static inline u32 rtw_read32(struct rtw_dev *rtwdev, u32 addr)
{
    return readl(rtwdev->mmio_base + addr);
}

static inline void rtw_write8(struct rtw_dev *rtwdev, u32 addr, u8 val)
{
    writeb(val, rtwdev->mmio_base + addr);
}

static inline void rtw_write16(struct rtw_dev *rtwdev, u32 addr, u16 val)
{
    writew(val, rtwdev->mmio_base + addr);
}

static inline void rtw_write32(struct rtw_dev *rtwdev, u32 addr, u32 val)
{
    writel(val, rtwdev->mmio_base + addr);
}

/* Escritura con máscara de bits */
static inline void rtw_write32_mask(struct rtw_dev *rtwdev, u32 addr,
                                     u32 mask, u32 data)
{
    u32 shift = __ffs(mask);
    u32 orig = rtw_read32(rtwdev, addr);
    u32 set = (orig & ~mask) | ((data << shift) & mask);
    rtw_write32(rtwdev, addr, set);
}
```

## 6. SECUENCIA DE POWER-ON / POWER-OFF

Existe una secuencia definida para Power On del RTL8822CE sobre PCIe y USB Bus, con parámetros de timing de PCIE PERST# específicos.

También existe una secuencia de Power Off definida para RTL8822CE-CG con parámetros de timing específicos.

### 6.1 Secuencia de Power On

```c
int rtw8822ce_power_on(struct rtw_dev *rtwdev)
{
    int ret;

    /* ============================================= */
    /* PASO 1: Verificar estado actual del chip      */
    /* ============================================= */

    /* Verificar que el chip responde */
    u32 val = rtw_read32(rtwdev, REG_SYS_CFG1);  /* 0x00FC */
    if (val == 0xFFFFFFFF)
        return -ENODEV;  /* Chip no responde */

    /* ============================================= */
    /* PASO 2: Pre-power on: Preparar reguladores    */
    /* ============================================= */

    /* Habilitar XTAL (cristal) */
    rtw_write8(rtwdev, REG_AFE_CTRL1 + 1, 0x82);    /* 0x001F */

    /* Configurar switching regulator */
    rtw_write32_mask(rtwdev, REG_SYS_SWR_CTRL1, 0x00C00000, 0x01); /* 0x000C */

    /* Habilitar LDO */
    rtw_write8(rtwdev, REG_LDO_SWR_CTRL, 0x01);  /* 0x0080 */

    /* ============================================= */
    /* PASO 3: Power on sequence                     */
    /* ============================================= */

    /* 3a. Habilitar funciones del sistema */
    u16 sys_func = rtw_read16(rtwdev, REG_SYS_FUNC_EN);  /* 0x0000 */
    sys_func |= BIT_FEN_MREGEN;       /* BIT(15) - MAC Register Enable */
    rtw_write16(rtwdev, REG_SYS_FUNC_EN, sys_func);

    /* 3b. Esperar a que el auto-load de EFUSE termine */
    ret = rtw_poll_reg(rtwdev, REG_SYS_EEPROM_CTRL, /* 0x0006 */
                       BIT_AUTOLOAD_SUS, BIT_AUTOLOAD_SUS, 1000);
    if (ret)
        return ret;

    /* 3c. Habilitar reloj MAC */
    rtw_write16(rtwdev, REG_SYS_CLK_CTRL,          /* 0x0004 */
                rtw_read16(rtwdev, REG_SYS_CLK_CTRL) |
                BIT_MAC_CLK_EN | BIT_ANA8M_EN);

    /* 3d. Habilitar bloque RF */
    rtw_write8(rtwdev, REG_RF_CTRL, 0x07);         /* 0x001C */
    /* RF_EN | RF_RSTB | RF_SDM_RSTB */

    /* 3e. Habilitar funciones de BB */
    sys_func = rtw_read16(rtwdev, REG_SYS_FUNC_EN);
    sys_func |= BIT_FEN_BBRSTB | BIT_FEN_BB_GLB_RST;
    rtw_write16(rtwdev, REG_SYS_FUNC_EN, sys_func);

    /* 3f. Habilitar CPU/Firmware */
    sys_func |= BIT_FEN_CPUEN;
    rtw_write16(rtwdev, REG_SYS_FUNC_EN, sys_func);

    /* ============================================= */
    /* PASO 4: Habilitar MAC subsystem               */
    /* ============================================= */

    /* Escribir Command Register - Habilitar TX/RX DMA y MAC */
    rtw_write32(rtwdev, REG_CR, 0x000004FF);        /* 0x0480 */
    /*  BIT_HCI_TXDMA_EN | BIT_HCI_RXDMA_EN |
        BIT_TXDMA_EN | BIT_RXDMA_EN |
        BIT_PROTOCOL_EN | BIT_SCHEDULE_EN |
        BIT_MAC_TX_EN | BIT_MAC_RX_EN |
        BIT_ENSEC */

    /* ============================================= */
    /* PASO 5: Verificar que el chip está ON          */
    /* ============================================= */

    val = rtw_read32(rtwdev, REG_CR);
    if (!(val & BIT_MAC_TX_EN) || !(val & BIT_MAC_RX_EN))
        return -EIO;

    return 0;
}
```

### 6.2 Secuencia de Power Off

```c
void rtw8822ce_power_off(struct rtw_dev *rtwdev)
{
    /* PASO 1: Parar TX */
    rtw_write8(rtwdev, REG_TXPAUSE, 0xFF);          /* 0x02F4 */

    /* PASO 2: Deshabilitar interrupciones */
    rtw_write32(rtwdev, REG_HIMR0, 0x00000000);     /* 0x0300 */
    rtw_write32(rtwdev, REG_HIMR1, 0x00000000);     /* 0x0308 */

    /* PASO 3: Deshabilitar MAC/DMA */
    rtw_write32(rtwdev, REG_CR, 0x00000000);         /* 0x0480 */

    /* PASO 4: Deshabilitar BB */
    u16 sys_func = rtw_read16(rtwdev, REG_SYS_FUNC_EN);
    sys_func &= ~(BIT_FEN_BBRSTB | BIT_FEN_BB_GLB_RST);
    rtw_write16(rtwdev, REG_SYS_FUNC_EN, sys_func); /* 0x0000 */

    /* PASO 5: Deshabilitar RF */
    rtw_write8(rtwdev, REG_RF_CTRL, 0x00);          /* 0x001C */

    /* PASO 6: Deshabilitar CPU */
    sys_func &= ~BIT_FEN_CPUEN;
    rtw_write16(rtwdev, REG_SYS_FUNC_EN, sys_func);

    /* PASO 7: Deshabilitar reloj MAC */
    rtw_write16(rtwdev, REG_SYS_CLK_CTRL,
                rtw_read16(rtwdev, REG_SYS_CLK_CTRL) & ~BIT_MAC_CLK_EN);

    /* PASO 8: Power down analógico */
    rtw_write8(rtwdev, REG_RSV_CTRL + 1, 0x00);     /* 0x0019 */

    /* PASO 9: Card disable (PCIe power state transition) */
    rtw_write8(rtwdev, REG_RSV_CTRL, 0x0E);         /* 0x0018 */
}
```

## 7. FIRMWARE: CARGA Y COMUNICACIÓN H2C/C2H

La firmware para RTL8822CE se encuentra en linux-firmware como `rtw88/rtw8822c_fw.bin`.

El procedimiento de inicialización contiene registros de secuencia de power on, descarga de firmware, carga de parámetros predefinidos, y registros específicos de mac/bb/rf.

### 7.1 Formato del Firmware


### 7.2 Procedimiento de Descarga de Firmware

```c
#define FW_HEADER_SIZE           64
#define FW_BLOCK_SIZE            4        /* Write 4 bytes at a time */
#define FW_PAGE_SIZE             4096
#define REG_MCUFW_CTRL           0x0090
#define REG_8051FW_CTRL          0x0080

struct rtw_fw_header {
    uint32_t signature;
    uint8_t  category;
    uint8_t  function;
    uint16_t version;
    uint8_t  subversion;
    uint8_t  subindex;
    uint8_t  reserved1[2];
    uint8_t  month;
    uint8_t  date;
    uint8_t  hour;
    uint8_t  minute;
    uint32_t ram_code_size;
    uint32_t reserved2;
    uint32_t dmem_addr;
    uint32_t imem_addr;
    uint32_t emem_addr;
    uint32_t emem_dmem_size;
    uint32_t iram_size;
    uint32_t eram_size;
    uint8_t  reserved3[8];
} __packed;

int rtw8822ce_download_firmware(struct rtw_dev *rtwdev,
                                 const uint8_t *fw_data, uint32_t fw_size)
{
    struct rtw_fw_header *hdr = (struct rtw_fw_header *)fw_data;
    const uint8_t *fw_body = fw_data + FW_HEADER_SIZE;
    uint32_t body_size = fw_size - FW_HEADER_SIZE;
    uint32_t val;
    int ret;

    /* Log firmware version */
    pr_info("RTL8822CE: Firmware version %u.%u.%u\n",
            hdr->version, hdr->subversion, hdr->subindex);

    /* ===== PASO 1: Preparar para descarga ===== */

    /* Habilitar CPU para download mode */
    val = rtw_read32(rtwdev, REG_MCUFW_CTRL);         /* 0x0090 */
    val |= BIT_MCUFWDL_EN;                             /* BIT(0) */
    val &= ~BIT_WINTINI_RDY;                           /* ~BIT(6) */
    rtw_write32(rtwdev, REG_MCUFW_CTRL, val);

    /* Seleccionar RAM download */
    val = rtw_read32(rtwdev, REG_MCUFW_CTRL);
    val |= BIT_RAM_DL_SEL;                             /* BIT(7) */
    rtw_write32(rtwdev, REG_MCUFW_CTRL, val);

    /* ===== PASO 2: Escribir firmware por páginas ===== */

    uint32_t offset = 0;
    uint32_t page = 0;

    while (offset < body_size) {
        /* Seleccionar página */
        uint8_t page_num = (uint8_t)(page & 0xFF);
        val = rtw_read32(rtwdev, REG_MCUFW_CTRL);
        val = (val & 0xFFFF00FF) | ((uint32_t)page_num << 8);
        rtw_write32(rtwdev, REG_MCUFW_CTRL, val);

        /* Escribir datos de firmware (en bloques de 4 bytes) */
        uint32_t write_size = min(body_size - offset, (uint32_t)FW_PAGE_SIZE);
        uint32_t i;
        for (i = 0; i < write_size; i += 4) {
            uint32_t word = 0;
            memcpy(&word, &fw_body[offset + i], min(4U, write_size - i));
            rtw_write32(rtwdev, 0x0180 + i, word);    /* FW download area */
        }

        offset += write_size;
        page++;
    }

    /* ===== PASO 3: Finalizar descarga ===== */

    /* Deshabilitar download mode */
    val = rtw_read32(rtwdev, REG_MCUFW_CTRL);
    val &= ~BIT_MCUFWDL_EN;
    rtw_write32(rtwdev, REG_MCUFW_CTRL, val);

    /* Reset CPU para ejecutar firmware */
    rtw_write8(rtwdev, REG_8051FW_CTRL + 2, 0x00);   /* 0x0082 */
    udelay(10);
    rtw_write8(rtwdev, REG_8051FW_CTRL + 2, 0x01);

    /* ===== PASO 4: Verificar que el firmware arrancó ===== */

    ret = rtw_poll_reg(rtwdev, REG_MCUFW_CTRL,
                       BIT_WINTINI_RDY, BIT_WINTINI_RDY, 10000);
    if (ret) {
        pr_err("RTL8822CE: Firmware failed to boot!\n");
        return -EBUSY;
    }

    /* Verificar FWDL check report */
    val = rtw_read32(rtwdev, REG_MCUFW_CTRL);
    if (val & BIT_FWDL_CHK_RPT)
        pr_info("RTL8822CE: Firmware download verified OK\n");

    return 0;
}
```

### 7.3 Comunicación H2C (Host-to-Chip)

```c
/*
 * H2C Command Format:
 * ┌─────────┬──────────┬──────────┬──────────┐
 * │ Byte 0  │ Byte 1-3 │ Byte 4-7 │ (ext)    │
 * │ CMD ID  │ Content  │ Extended │          │
 * └─────────┴──────────┴──────────┴──────────┘
 *
 * CMD ID [7:0]:
 *   [6:0] = Sub-command ID
 *   [7]   = Class bit (0=common, 1=special)
 */

/* H2C Command IDs */
#define H2C_CMD_RSVD_PAGE          0x00
#define H2C_CMD_MEDIA_STATUS_RPT   0x01
#define H2C_CMD_SET_PWR_MODE       0x20
#define H2C_CMD_PS_TUNING_PARA     0x21
#define H2C_CMD_RA_INFO            0x40
#define H2C_CMD_BT_MP_OPER         0x67
#define H2C_CMD_BT_WIFI_CTRL       0x69
#define H2C_CMD_QUERY_BT_INFO      0x60
#define H2C_CMD_WL_CH_INFO         0x66
#define H2C_CMD_COEX_TDMA          0x60
#define H2C_CMD_LPS_PG_INFO        0x2B
#define H2C_CMD_DEFAULT_PORT       0x2C
#define H2C_CMD_BCN_OFFLOAD        0x22

/* Función para enviar H2C */
#define REG_HMETFR    0x01CC

int rtw_fw_send_h2c_command(struct rtw_dev *rtwdev, uint8_t *h2c_cmd)
{
    uint32_t box_reg;
    uint32_t box_ext_reg;
    uint8_t box;
    uint32_t val;
    int ret;

    /* Obtener el mailbox actual */
    box = rtwdev->h2c_box_num;

    /* Seleccionar registros de mailbox */
    switch (box) {
    case 0:
        box_reg     = REG_HMEBOX0;     /* 0x01D0 */
        box_ext_reg = 0x0098;           /* REG_HMEBOX_E0_E1 */
        break;
    case 1:
        box_reg     = REG_HMEBOX1;     /* 0x01D4 */
        box_ext_reg = 0x009A;
        break;
    case 2:
        box_reg     = REG_HMEBOX2;     /* 0x01D8 */
        box_ext_reg = 0x009C;
        break;
    case 3:
        box_reg     = REG_HMEBOX3;     /* 0x01DC */
        box_ext_reg = 0x009E;
        break;
    default:
        return -EINVAL;
    }

    /* Verificar que el mailbox previo fue procesado */
    ret = rtw_poll_reg(rtwdev, REG_HMETFR, BIT(box), 0, 1000);
    if (ret) {
        pr_err("H2C box %d not ready\n", box);
        return ret;
    }

    /* Escribir extensión primero (bytes 4-7) */
    val = le32_to_cpu(*(uint32_t *)&h2c_cmd[4]);
    rtw_write32(rtwdev, box_ext_reg, val);

    /* Escribir comando (bytes 0-3) - esto dispara el envío */
    val = le32_to_cpu(*(uint32_t *)&h2c_cmd[0]);
    rtw_write32(rtwdev, box_reg, val);

    /* Avanzar al siguiente mailbox */
    rtwdev->h2c_box_num = (box + 1) % 4;

    return 0;
}
```

### 7.4 Comunicación C2H (Chip-to-Host)

```c
/*
 * C2H Event Format (recibido como paquete RX especial):
 * ┌──────────┬──────────┬──────────────────┬──────────┐
 * │ Byte 0   │ Byte 1   │ Byte 2~(N-1)    │ Byte N   │
 * │ Event ID │ Seq Num  │ Event Payload    │ Trigger  │
 * └──────────┴──────────┴──────────────────┴──────────┘
 */

/* C2H Event IDs */
#define C2H_EVT_DBG                0x00
#define C2H_EVT_SURVEY             0x01
#define C2H_EVT_SURVEY_DONE        0x02
#define C2H_EVT_TX_REPORT          0x03
#define C2H_EVT_BT_INFO            0x09
#define C2H_EVT_BT_MP_INFO         0x0B
#define C2H_EVT_RA_RPT             0x0C
#define C2H_EVT_SPECIAL_PKT        0x0E
#define C2H_EVT_COEX               0x10
#define C2H_EVT_SCAN_RESULT        0x14
#define C2H_EVT_ADAPTIVITY          0x22
#define C2H_EVT_WOW_WAKEUP         0x24

void rtw_fw_c2h_handler(struct rtw_dev *rtwdev, uint8_t *c2h_data, uint32_t len)
{
    uint8_t event_id = c2h_data[0];
    uint8_t seq      = c2h_data[1];
    uint8_t *payload = &c2h_data[2];
    uint32_t plen    = len - 2;

    switch (event_id) {
    case C2H_EVT_TX_REPORT:
        rtw_handle_tx_report(rtwdev, payload, plen);
        break;
    case C2H_EVT_BT_INFO:
        rtw_coex_bt_info_handler(rtwdev, payload, plen);
        break;
    case C2H_EVT_RA_RPT:
        rtw_handle_ra_report(rtwdev, payload, plen);
        break;
    case C2H_EVT_COEX:
        rtw_coex_c2h_handler(rtwdev, payload, plen);
        break;
    default:
        pr_debug("RTL8822CE: Unknown C2H event 0x%02X\n", event_id);
        break;
    }
}
```

### 8.3 TX Ring Management

```c
#define RTW_TX_RING_SIZE    256
#define RTW_TX_DESC_SIZE    40

struct rtw_tx_ring {
    /* TX Buffer Descriptor (BD) Ring */
    struct rtw_tx_bd *bd;          /* Virtual address */
    dma_addr_t        bd_dma;      /* Physical (DMA) address */
    uint32_t          bd_count;    /* Number of BDs in ring */

    /* Write/Read pointers */
    uint32_t          wr_idx;      /* Host write index */
    uint32_t          rd_idx;      /* HW read index (from register) */

    /* SKB tracking */
    struct sk_buff   *skbs[RTW_TX_RING_SIZE];
};

/* TX Buffer Descriptor (16 bytes cada uno) */
struct rtw_tx_bd {
    uint16_t buf_size;
    uint16_t flags;
    uint32_t phy_addr_low;     /* DMA address lower 32-bit */
    uint32_t phy_addr_high;    /* DMA address upper 32-bit */
    uint32_t reserved;
} __packed;

/* Inicializar TX ring */
int rtw_pci_init_tx_ring(struct rtw_dev *rtwdev, int queue_id)
{
    struct rtw_tx_ring *ring = &rtwdev->tx_ring[queue_id];
    size_t size = RTW_TX_RING_SIZE * sizeof(struct rtw_tx_bd);

    /* Allocar ring de descriptores (DMA coherent) */
    ring->bd = dma_alloc_coherent(rtwdev->dev, size,
                                   &ring->bd_dma, GFP_KERNEL);
    if (!ring->bd)
        return -ENOMEM;

    memset(ring->bd, 0, size);
    ring->bd_count = RTW_TX_RING_SIZE;
    ring->wr_idx = 0;
    ring->rd_idx = 0;

    /* Programar dirección del ring en hardware */
    uint32_t addr_reg, addr_h_reg, num_reg;
    switch (queue_id) {
    case RTW_TX_QUEUE_VO:
        addr_reg   = 0x0310;  /* REG_VOQ_TXBD_DESA */
        addr_h_reg = 0x0314;
        num_reg    = 0x0318;
        break;
    case RTW_TX_QUEUE_VI:
        addr_reg   = 0x0320;
        addr_h_reg = 0x0324;
        num_reg    = 0x0328;
        break;
    case RTW_TX_QUEUE_BE:
        addr_reg   = 0x0330;
        addr_h_reg = 0x0334;
        num_reg    = 0x0338;
        break;
    case RTW_TX_QUEUE_BK:
        addr_reg   = 0x0340;
        addr_h_reg = 0x0344;
        num_reg    = 0x0348;
        break;
    case RTW_TX_QUEUE_MGT:
        addr_reg   = 0x0350;
        addr_h_reg = 0x0354;
        num_reg    = 0x0358;
        break;
    case RTW_TX_QUEUE_HI:
        addr_reg   = 0x0370;
        addr_h_reg = 0x0374;
        num_reg    = 0x0378;
        break;
    case RTW_TX_QUEUE_BCN:
        addr_reg   = 0x0380;
        addr_h_reg = 0x0384;
        num_reg    = 0;        /* BCN queue no tiene num reg */
        break;
    default:
        return -EINVAL;
    }

    rtw_write32(rtwdev, addr_reg,   (uint32_t)(ring->bd_dma & 0xFFFFFFFF));
    rtw_write32(rtwdev, addr_h_reg, (uint32_t)(ring->bd_dma >> 32));
    if (num_reg)
        rtw_write16(rtwdev, num_reg, RTW_TX_RING_SIZE);

    return 0;
}

/* Transmitir un paquete */
int rtw_pci_tx(struct rtw_dev *rtwdev, struct sk_buff *skb, int queue_id)
{
    struct rtw_tx_ring *ring = &rtwdev->tx_ring[queue_id];
    struct rtw_tx_bd *bd;
    dma_addr_t dma_addr;
    uint32_t idx;

    /* Verificar espacio disponible */
    idx = ring->wr_idx;
    if (((idx + 1) % ring->bd_count) == ring->rd_idx)
        return -ENOSPC;  /* Ring full */

    /* Prepend TX descriptor al SKB */
    /* (ya debe estar prepended por la capa superior) */

    /* Mapear buffer para DMA */
    dma_addr = dma_map_single(rtwdev->dev, skb->data, skb->len,
                               DMA_TO_DEVICE);
    if (dma_mapping_error(rtwdev->dev, dma_addr))
        return -ENOMEM;

    /* Llenar BD */
    bd = &ring->bd[idx];
    bd->buf_size       = skb->len;
    bd->flags          = 0;  /* OWN bit, etc. */
    bd->phy_addr_low   = (uint32_t)(dma_addr & 0xFFFFFFFF);
    bd->phy_addr_high  = (uint32_t)(dma_addr >> 32);

    /* Guardar referencia al SKB */
    ring->skbs[idx] = skb;

    /* Avanzar write pointer */
    ring->wr_idx = (idx + 1) % ring->bd_count;

    /* Notificar al hardware: escribir write pointer */
    uint32_t idx_reg;
    switch (queue_id) {
    case RTW_TX_QUEUE_VO:  idx_reg = 0x03B8; break;
    case RTW_TX_QUEUE_VI:  idx_reg = 0x03BC; break;
    case RTW_TX_QUEUE_BE:  idx_reg = 0x03C0; break;
    case RTW_TX_QUEUE_BK:  idx_reg = 0x03C4; break;
    case RTW_TX_QUEUE_MGT: idx_reg = 0x03C8; break;
    case RTW_TX_QUEUE_HI:  idx_reg = 0x03D0; break;
    case RTW_TX_QUEUE_BCN: idx_reg = 0x03D4; break;
    default: return -EINVAL;
    }
    rtw_write32(rtwdev, idx_reg, ring->wr_idx);

    return 0;
}
```

### 9.2 RX Ring Management

```c
#define RTW_RX_RING_SIZE    512
#define RTW_RX_BUF_SIZE     16384   /* 16 KB per buffer */

struct rtw_rx_ring {
    struct rtw_rx_bd *bd;
    dma_addr_t        bd_dma;
    uint32_t          bd_count;
    uint32_t          rd_idx;

    /* Pre-allocated buffers */
    struct {
        void     *vaddr;
        dma_addr_t dma;
    } bufs[RTW_RX_RING_SIZE];
};

/* RX Buffer Descriptor */
struct rtw_rx_bd {
    uint16_t buf_size;
    uint16_t flags;          /* [15] OWN, [14] EOR */
    uint32_t phy_addr_low;
    uint32_t phy_addr_high;
    uint32_t total_len;
} __packed;

int rtw_pci_init_rx_ring(struct rtw_dev *rtwdev)
{
    struct rtw_rx_ring *ring = &rtwdev->rx_ring;
    size_t desc_size = RTW_RX_RING_SIZE * sizeof(struct rtw_rx_bd);

    ring->bd = dma_alloc_coherent(rtwdev->dev, desc_size,
                                   &ring->bd_dma, GFP_KERNEL);
    if (!ring->bd)
        return -ENOMEM;

    ring->bd_count = RTW_RX_RING_SIZE;
    ring->rd_idx = 0;

    /* Pre-allocar buffers */
    for (int i = 0; i < RTW_RX_RING_SIZE; i++) {
        ring->bufs[i].vaddr = dma_alloc_coherent(rtwdev->dev,
                                                   RTW_RX_BUF_SIZE,
                                                   &ring->bufs[i].dma,
                                                   GFP_KERNEL);
        if (!ring->bufs[i].vaddr)
            goto cleanup;

        /* Configurar BD */
        ring->bd[i].buf_size      = RTW_RX_BUF_SIZE;
        ring->bd[i].flags         = (1 << 15);  /* OWN = 1 (HW owns) */
        ring->bd[i].phy_addr_low  = (uint32_t)(ring->bufs[i].dma & 0xFFFFFFFF);
        ring->bd[i].phy_addr_high = (uint32_t)(ring->bufs[i].dma >> 32);

        /* Marcar End of Ring en el último BD */
        if (i == RTW_RX_RING_SIZE - 1)
            ring->bd[i].flags |= (1 << 14);  /* EOR */
    }

    /* Programar ring address en hardware */
    rtw_write32(rtwdev, 0x0360, (uint32_t)(ring->bd_dma & 0xFFFFFFFF));
    rtw_write32(rtwdev, 0x0364, (uint32_t)(ring->bd_dma >> 32));
    rtw_write16(rtwdev, 0x0368, RTW_RX_RING_SIZE);

    return 0;

cleanup:
    /* Limpiar buffers ya alocados */
    for (int j = 0; j < i; j++)
        dma_free_coherent(rtwdev->dev, RTW_RX_BUF_SIZE,
                          ring->bufs[j].vaddr, ring->bufs[j].dma);
    dma_free_coherent(rtwdev->dev, desc_size, ring->bd, ring->bd_dma);
    return -ENOMEM;
}

/* Procesar paquete RX */
void rtw_pci_rx_handler(struct rtw_dev *rtwdev)
{
    struct rtw_rx_ring *ring = &rtwdev->rx_ring;

    while (1) {
        struct rtw_rx_bd *bd = &ring->bd[ring->rd_idx];

        /* Verificar si hay paquete disponible (OWN = 0 → host owns) */
        if (bd->flags & (1 << 15))
            break;  /* HW aún tiene este BD */

        uint32_t pkt_len = bd->total_len;
        void *buf = ring->bufs[ring->rd_idx].vaddr;

        /* Parsear RX descriptor */
        uint32_t *rxdesc = (uint32_t *)buf;
        uint16_t total_len   = (rxdesc[0] >> 16) & 0x3FFF;
        uint8_t  drv_info_sz = ((rxdesc[0] >> 14) & 0x03) * 8;
        bool     phy_status  = !!(rxdesc[0] & (1 << 13));
        bool     crc_error   = !!(rxdesc[0] & (1 << 7));
        bool     icv_error   = !!(rxdesc[0] & (1 << 6));
        uint8_t  shift       = (rxdesc[0] >> 8) & 0x07;

        uint32_t desc_sz = 24;  /* RX descriptor size */
        uint8_t *frame_data = (uint8_t *)buf + desc_sz + drv_info_sz + shift;
        uint32_t frame_len  = total_len - desc_sz - drv_info_sz - shift;

        if (!crc_error && !icv_error) {
            /* Entregar frame al stack de red */
            rtw_rx_deliver(rtwdev, frame_data, frame_len,
                          phy_status ? (buf + desc_sz) : NULL,
                          drv_info_sz);
        }

        /* Devolver BD al hardware */
        bd->flags = (1 << 15);  /* OWN = 1 */
        if (ring->rd_idx == RTW_RX_RING_SIZE - 1)
            bd->flags |= (1 << 14);  /* EOR */

        ring->rd_idx = (ring->rd_idx + 1) % ring->bd_count;

        /* Actualizar RX read pointer */
        rtw_write32(rtwdev, 0x03CC, ring->rd_idx);
    }
}
```

### 10.1 Acceso a EFUSE (continuación)

```c
int rtw_efuse_read_byte(struct rtw_dev *rtwdev, uint16_t addr, uint8_t *data)
{
    uint32_t val;
    int ret;

    /* Habilitar acceso a EFUSE */
    rtw_write8(rtwdev, 0x00CF, 0x69);  /* REG_EFUSE_ACCESS = unlock */

    /* Escribir dirección y trigger lectura */
    val = rtw_read32(rtwdev, 0x0028);  /* REG_EFUSE_CTRL */
    val &= ~(0x3FF << 16);            /* Limpiar campo de dirección */
    val |= ((addr & 0x3FF) << 16);    /* Escribir dirección [25:16] */
    val &= ~BIT(31);                   /* Bit 31 = 0: lectura */
    rtw_write32(rtwdev, 0x0028, val);

    /* Esperar a que la lectura termine (bit 31 → 1 cuando ready) */
    ret = rtw_poll_reg(rtwdev, 0x0028, BIT(31), BIT(31), 5000);
    if (ret) {
        pr_err("EFUSE read timeout at addr 0x%04X\n", addr);
        return ret;
    }

    /* Leer dato */
    val = rtw_read32(rtwdev, 0x0028);
    *data = (uint8_t)(val & 0xFF);

    /* Cerrar acceso EFUSE */
    rtw_write8(rtwdev, 0x00CF, 0x00);

    return 0;
}

/* Leer el mapa lógico completo de EFUSE */
int rtw_efuse_read_logical_map(struct rtw_dev *rtwdev, uint8_t *map, uint32_t map_size)
{
    /*
     * EFUSE usa un formato comprimido:
     * - Header byte: identifica sección (4-bit section id + valid bits)
     * - Extended header para secciones > 15
     * - Data bytes: 1-8 bytes de datos por sección
     *
     * Formato header normal (1 byte):
     *   [7:4] = Section ID (0x0 - 0xF)
     *   [3:0] = Word Enable (bit=0 → word presente, bit=1 → word ausente)
     *
     * Formato extended header (2 bytes):
     *   Byte1: [7:4] = 0xF (marca extended), [3:0] = Section ID bits [3:0]
     *   Byte2: [7:4] = Section ID bits [7:4], [3:0] = Word Enable
     */

    memset(map, 0xFF, map_size);  /* Inicializar con 0xFF */

    uint16_t efuse_addr = 0;
    uint8_t header, word_en;
    uint16_t section;

    while (efuse_addr < EFUSE_RAW_SIZE) {
        uint8_t byte;
        int ret = rtw_efuse_read_byte(rtwdev, efuse_addr++, &byte);
        if (ret)
            return ret;

        /* 0xFF indica fin del mapa EFUSE */
        if (byte == 0xFF)
            break;

        header = byte;

        if ((header & 0x1F) == 0x0F) {
            /* Extended header */
            section = (header >> 5) & 0x07;  /* bits [7:5] → section[2:0] */

            ret = rtw_efuse_read_byte(rtwdev, efuse_addr++, &byte);
            if (ret)
                return ret;

            section |= ((byte >> 4) & 0x0F) << 3;  /* bits [7:4] → section[6:3] */
            word_en = byte & 0x0F;
        } else {
            /* Normal header */
            section = (header >> 4) & 0x0F;
            word_en = header & 0x0F;
        }

        /* Offset base en el mapa lógico */
        uint16_t base = section * 8;

        /* Leer words (cada word = 2 bytes) */
        for (int i = 0; i < 4; i++) {
            if (!(word_en & BIT(i))) {
                /* Word i está presente en EFUSE */
                uint16_t offset = base + (i * 2);

                uint8_t d0, d1;
                ret = rtw_efuse_read_byte(rtwdev, efuse_addr++, &d0);
                if (ret) return ret;
                ret = rtw_efuse_read_byte(rtwdev, efuse_addr++, &d1);
                if (ret) return ret;

                if (offset < map_size)
                    map[offset] = d0;
                if (offset + 1 < map_size)
                    map[offset + 1] = d1;
            }
        }
    }

    return 0;
}
```

## 11. GESTIÓN DE ENERGÍA (Power Management)

### 11.1 Power States del RTL8822CE

```
┌─────────────┐     ┌──────────────┐     ┌──────────────┐
│  D0 (Active) │────▶│  D1 (Sleep)  │────▶│  D3 (Off)    │
│  Full Power  │◀────│  LPS / IPS   │◀────│  Radio Off   │
└──────┬──────┘     └──────────────┘     └──────────────┘
       │
       ▼
┌──────────────┐
│  D2 (Doze)   │
│  Low Power   │
└──────────────┘

LPS = Leisure Power Save (connected idle)
IPS = Inactive Power Save (disconnected)
```

### 11.2 Implementación LPS (Leisure Power Save)

```c
/* ═══════════════════════════════════════════════════ */
/*        REGISTROS DE POWER MANAGEMENT               */
/* ═══════════════════════════════════════════════════ */
#define REG_PCIE_HRPWM       0x03F2   /* Host Request Power Mode */
#define REG_PCIE_HCPWM       0x03F3   /* Host Confirm Power Mode */
#define REG_PCIE_HRPWM2      0x03F8   /* Host Request Power Mode 2 */
#define REG_PCIE_HCPWM2      0x03FA   /* Host Confirm Power Mode 2 */

/* Power mode bits */
#define PS_MODE_ACTIVE        0x00
#define PS_MODE_MIN           0x01
#define PS_MODE_MAX           0x02
#define PS_MODE_DTIM          0x03
#define PS_MODE_VOIP          0x04
#define PS_MODE_UAPSD_WMM     0x05

/* H2C command para set power mode */
struct h2c_pwr_mode {
    uint8_t cmd_id;        /* H2C_CMD_SET_PWR_MODE = 0x20 */
    uint8_t mode;          /* PS_MODE_xxx */
    uint8_t clk_req;       /* Clock Request: 0=disable, 1=enable */
    uint8_t rlbm;          /* Rate Limiting Beacon Mode */
    uint8_t smart_ps;      /* Smart Power Save Mode */
    uint8_t awake_interval; /* Awake Interval (beacon units) */
    uint8_t all_queue_uapsd; /* All Queue UAPSD */
    uint8_t pwr_state;     /* Power State */
} __packed;

int rtw_pci_enter_lps(struct rtw_dev *rtwdev, uint8_t mode)
{
    struct h2c_pwr_mode cmd = {0};

    /* Configurar comando H2C de power mode */
    cmd.cmd_id         = 0x20;  /* H2C_CMD_SET_PWR_MODE */
    cmd.mode           = mode;
    cmd.clk_req        = 1;     /* Mantener clock request activo */
    cmd.rlbm           = 2;     /* Rate limit by DTIM */
    cmd.smart_ps       = 2;     /* Smart PS mode 2 */
    cmd.awake_interval = 1;     /* Despertar cada beacon */

    /* Enviar H2C */
    int ret = rtw_fw_send_h2c_command(rtwdev, (uint8_t *)&cmd);
    if (ret)
        return ret;

    /* Configurar RPWM (Request Power Mode) para el hardware */
    uint8_t rpwm_val = 0;
    switch (mode) {
    case PS_MODE_MIN:
    case PS_MODE_MAX:
    case PS_MODE_DTIM:
        rpwm_val = 0x02;  /* Request sleep */
        break;
    default:
        rpwm_val = 0x0C;  /* Active */
        break;
    }

    /* Toggle bit [7] para indicar nuevo request */
    rpwm_val |= (rtwdev->rpwm_seq++ & 0x01) ? BIT(7) : 0;

    rtw_write8(rtwdev, REG_PCIE_HRPWM, rpwm_val);  /* 0x03F2 */

    return 0;
}

int rtw_pci_leave_lps(struct rtw_dev *rtwdev)
{
    struct h2c_pwr_mode cmd = {0};

    /* Despertar: escribir RPWM primero */
    uint8_t rpwm_val = 0x0C;  /* Active mode */
    rpwm_val |= (rtwdev->rpwm_seq++ & 0x01) ? BIT(7) : 0;
    rtw_write8(rtwdev, REG_PCIE_HRPWM, rpwm_val);

    /* Esperar confirmación */
    int ret = rtw_poll_reg(rtwdev, REG_PCIE_HCPWM, 0x0C, 0x0C, 5000);
    if (ret)
        pr_warn("LPS leave: HCPWM timeout\n");

    /* Enviar H2C active mode */
    cmd.cmd_id = 0x20;
    cmd.mode   = PS_MODE_ACTIVE;
    return rtw_fw_send_h2c_command(rtwdev, (uint8_t *)&cmd);
}
```

### 11.3 IPS (Inactive Power Save)

```c
int rtw_enter_ips(struct rtw_dev *rtwdev)
{
    /* PASO 1: Deshabilitar TX */
    rtw_write8(rtwdev, 0x02F4, 0xFF);         /* REG_TXPAUSE: pausar todo */

    /* PASO 2: Deshabilitar RF */
    rtw_write8(rtwdev, 0x001C, 0x00);         /* REG_RF_CTRL: RF off */

    /* PASO 3: Deshabilitar BB */
    u16 sys_func = rtw_read16(rtwdev, 0x0000);
    sys_func &= ~(BIT(0) | BIT(1));           /* FEN_BBRSTB | FEN_BB_GLB_RST */
    rtw_write16(rtwdev, 0x0000, sys_func);

    /* PASO 4: Deshabilitar MAC clock */
    u16 clk = rtw_read16(rtwdev, 0x0004);
    clk &= ~BIT(11);                           /* ~BIT_MAC_CLK_EN */
    rtw_write16(rtwdev, 0x0004, clk);

    /* PASO 5: Bajar PCIe a D3 si es posible */
    rtw_pci_set_power_state(rtwdev, PCI_D3hot);

    rtwdev->power_state = RTW_POWER_IPS;
    return 0;
}

int rtw_leave_ips(struct rtw_dev *rtwdev)
{
    /* Revertir toda la secuencia */
    rtw_pci_set_power_state(rtwdev, PCI_D0);

    /* Re-ejecutar power on sequence */
    int ret = rtw8822ce_power_on(rtwdev);
    if (ret)
        return ret;

    /* Re-cargar firmware */
    ret = rtw8822ce_download_firmware(rtwdev,
                                      rtwdev->fw_data,
                                      rtwdev->fw_size);
    if (ret)
        return ret;

    /* Re-inicializar MAC/BB/RF */
    rtw8822ce_init_mac_reg(rtwdev);
    rtw8822ce_init_bb_reg(rtwdev);
    rtw8822ce_init_rf_reg(rtwdev);

    rtwdev->power_state = RTW_POWER_ACTIVE;
    return 0;
}
```

### 11.4 PCIe L1/L1.2 Power States

```c
/* ═══════════════════════════════════════════════════ */
/*        REGISTROS PCIe LINK POWER MANAGEMENT         */
/* ═══════════════════════════════════════════════════ */

/* PCIe Config Space — PM Capability (offset 0x40) */
#define PCI_PM_CAP          0x40
#define PCI_PM_CTRL         0x44
#define PCI_PM_CTRL_STATE   0x0003    /* Mask for power state */
#define PCI_D0              0x00
#define PCI_D1              0x01
#define PCI_D2              0x02
#define PCI_D3hot           0x03

/* PCIe Link Control Register (en PCIe Capability) */
#define PCIE_LINK_CTRL      0x80      /* Offset puede variar */
#define PCIE_LINK_L0S_EN    BIT(0)
#define PCIE_LINK_L1_EN     BIT(1)
#define PCIE_LINK_CLK_REQ   BIT(8)

/* Realtek-specific ASPM control */
#define REG_PCIE_PS_CTRL      0x0719
#define BIT_EN_CLK_REQ         BIT(4)
#define BIT_EN_ASPM_L1         BIT(3)
#define BIT_EN_ASPM_L0S        BIT(2)
#define BIT_DIS_L1_REENTRY     BIT(7)

void rtw_pci_configure_aspm(struct rtw_dev *rtwdev, bool enable_l1)
{
    uint8_t val;

    /* Configurar ASPM en el lado Realtek */
    val = rtw_read8(rtwdev, REG_PCIE_PS_CTRL);  /* 0x0719 */

    if (enable_l1) {
        val |= BIT_EN_ASPM_L1;      /* Habilitar L1 */
        val |= BIT_EN_CLK_REQ;      /* Habilitar Clock Request */
        val &= ~BIT_DIS_L1_REENTRY; /* Permitir re-entry a L1 */
    } else {
        val &= ~BIT_EN_ASPM_L1;
        val &= ~BIT_EN_ASPM_L0S;
        val &= ~BIT_EN_CLK_REQ;
    }

    rtw_write8(rtwdev, REG_PCIE_PS_CTRL, val);

    /* Configurar backdoor access para L1 substates */
    if (enable_l1) {
        /* Habilitar L1.1 y L1.2 via DBI */
        rtw_dbi_write8(rtwdev, 0x0718, 0x77);  /* L1 substates enable */

        /* LTR (Latency Tolerance Reporting) */
        rtw_write32(rtwdev, 0x0798, 0x883C883C); /* LTR idle latency */
        rtw_write32(rtwdev, 0x079C, 0x880B880B); /* LTR active latency */
    }
}

/* DBI (Debug Bus Interface) — acceso a PCIe config space vía MMIO */
void rtw_dbi_write8(struct rtw_dev *rtwdev, uint16_t addr, uint8_t data)
{
    uint32_t val;

    /* Escribir datos en DBI_WDATA */
    uint16_t addr_aligned = addr & ~0x03;
    uint8_t  byte_offset  = addr & 0x03;

    rtw_write32(rtwdev, 0x0204, (uint32_t)data << (byte_offset * 8));

    /* Configurar flag: write + byte enable + address */
    val  = BIT(16);                              /* Write flag */
    val |= (1 << (byte_offset + 12));           /* Byte enable */
    val |= (addr_aligned & 0x0FFF);             /* Address */
    rtw_write32(rtwdev, 0x020C, val);

    /* Wait for completion */
    rtw_poll_reg(rtwdev, 0x020C, BIT(16), 0, 1000);
}

uint8_t rtw_dbi_read8(struct rtw_dev *rtwdev, uint16_t addr)
{
    uint32_t val;

    uint16_t addr_aligned = addr & ~0x03;
    uint8_t  byte_offset  = addr & 0x03;

    /* Configurar flag: read + byte enable + address */
    val  = 0;                                    /* Read flag (bit16=0) */
    val |= (0x0F << 12);                        /* All bytes enable */
    val |= (addr_aligned & 0x0FFF);
    rtw_write32(rtwdev, 0x020C, val);

    /* Wait for completion */
    rtw_poll_reg(rtwdev, 0x020C, BIT(16), BIT(16), 1000);

    val = rtw_read32(rtwdev, 0x0208);            /* DBI_RDATA */
    return (uint8_t)(val >> (byte_offset * 8));
}
```

```c
/* Configurar TDMA de coexistencia vía firmware */
struct h2c_coex_tdma {

/* Perfiles de coexistencia */

void rtw_coex_set_tdma(struct rtw_dev *rtwdev, uint8_t type,
                        uint8_t interval, uint8_t wl_duration,
                        uint8_t bt_duration)
{

    cmd.cmd_id = 0x60;
    cmd.type   = type;
    cmd.byte1  = interval;
    cmd.byte2  = wl_duration;
    cmd.byte3  = bt_duration;
    cmd.byte4  = 0x50;           /* Default switches */

    rtw_fw_send_h2c_command(rtwdev, (uint8_t *)&cmd);
}

/* Aplicar perfil de coexistencia según escenario */
void rtw_coex_apply_scenario(struct rtw_dev *rtwdev,
                              enum rtw_coex_scenario scenario)
{
    switch (scenario) {
    case COEX_SCENARIO_IDLE:
        /* Sin actividad BT: WiFi tiene prioridad total */
        rtw_coex_set_table(rtwdev,
                           0xFFFFFFFF,    /* WiFi siempre permitido */
                           0x00000000,    /* BT no necesita slots */
                           0x00000000);
        rtw_coex_set_tdma(rtwdev, COEX_TDMA_OFF, 0, 0, 0);
        break;

    case COEX_SCENARIO_A2DP:
        /* Audio BT: TDMA con 60% WiFi, 40% BT */
        rtw_coex_set_table(rtwdev,
                           0xAAAA5555,
                           0x5555AAAA,
                           0xFFFF);
        rtw_coex_set_tdma(rtwdev, COEX_TDMA_FIX_SLOT, 100, 60, 40);
        break;

    case COEX_SCENARIO_HID:
        /* HID: baja latencia BT necesaria */
        rtw_coex_set_table(rtwdev,
                           0xAAAA5555,
                           0x5A5A5A5A,
                           0xFFFF);
        rtw_coex_set_tdma(rtwdev, COEX_TDMA_FLEX_SLOT, 50, 30, 20);
        break;

    case COEX_SCENARIO_SCO:
        /* Llamada de voz: BT tiene alta prioridad */
        rtw_coex_set_table(rtwdev,
                           0x55555555,    /* WiFi cede más slots */
                           0xAAAAAAAA,    /* BT prioridad */
                           0xFFFF);
        rtw_coex_set_tdma(rtwdev, COEX_TDMA_FIX_SLOT, 50, 15, 35);
        break;

    default:
        /* Modo genérico compartido */
        rtw_coex_set_table(rtwdev,
                           0x66555555,
                           0x66555555,
                           0xFFFF);
        rtw_coex_set_tdma(rtwdev, COEX_TDMA_AUTO, 100, 50, 50);
        break;
    }
}

/* Handler de info BT (llamado desde C2H) */
void rtw_coex_bt_info_handler(struct rtw_dev *rtwdev,
                               uint8_t *payload, uint32_t len)
{
    /* payload[0]: BT link info
     *   BIT(0): SCO exist
     *   BIT(1): A2DP exist
     *   BIT(2): HID exist
     *   BIT(3): PAN exist
     *
     * payload[1]: BT retry count
     * payload[2]: BT RSSI
     * payload[3]: BT status
     *   0x00: idle
     *   0x01: inquiry/page scan
     *   0x02: connected idle
     *   0x03: ACL busy
     *   0x04: SCO busy
     */

    uint8_t bt_link = payload[0];
    uint8_t bt_rssi = payload[2];
    uint8_t bt_status = payload[3];

    /* Determinar escenario */
    enum rtw_coex_scenario scenario = COEX_SCENARIO_IDLE;

    if (bt_link & BIT(0))       /* SCO */
        scenario = COEX_SCENARIO_SCO;
    else if (bt_link & BIT(1))  /* A2DP */
        scenario = COEX_SCENARIO_A2DP;
    else if (bt_link & BIT(2))  /* HID */
        scenario = COEX_SCENARIO_HID;
    else if (bt_link & BIT(3))  /* PAN */
        scenario = COEX_SCENARIO_PAN;

    rtw_coex_apply_scenario(rtwdev, scenario);
}
```

### 13.1 IQ Calibration (IQK)

```c
/* ═══════════════════════════════════════════════════ */
/*         IQ CALIBRATION (IQK) REGISTERS              */
/* ═══════════════════════════════════════════════════ */

/* IQK se ejecuta para corregir el desbalance I/Q del transceiver */

/* BB Registers para IQK */
#define REG_IQK_AGC_PTS          0x0E60
#define REG_IQK_AGC_RSP          0x0E64
#define REG_TX_IQK               0x0E68
#define REG_RX_IQK               0x0E6C
#define REG_IQK_TONE_A           0x0E70
#define REG_IQK_TONE_B           0x0E74
#define REG_IQK_RES_TX           0x0E78
#define REG_IQK_RES_RX           0x0E7C

/* RF Register para IQK */
#define RF_IQK_REG               0x43

/* Trigger IQK */
#define REG_IQK_CTRL             0x0E28
#define BIT_IQK_START            BIT(0)

/* Estado de IQK */
#define REG_IQK_STATUS           0x0EA0
#define BIT_IQK_DONE             BIT(0)
#define BIT_IQK_TX_FAIL          BIT(1)
#define BIT_IQK_RX_FAIL          BIT(2)

/* Resultados de IQK */
#define REG_IQK_TX_X             0x0E94    /* TX I result */
#define REG_IQK_TX_Y             0x0E9C    /* TX Q result */
#define REG_IQK_RX_X             0x0EA4    /* RX I result */
#define REG_IQK_RX_Y             0x0EAC    /* RX Q result */

int rtw8822ce_iqk(struct rtw_dev *rtwdev)
{
    uint32_t val;
    int ret;

    pr_info("RTL8822CE: Starting IQ Calibration\n");

    /* ===== PASO 1: Backup registros ===== */
    uint32_t bb_backup[] = {
        rtw_read32(rtwdev, 0x0800),    /* REG_FPGA0_RFMOD */
        rtw_read32(rtwdev, 0x0900),    /* REG_OFDM0_TRX_PATH */
        rtw_read32(rtwdev, 0x0C00),    /* REG_OFDM1_LSTF */
        rtw_read32(rtwdev, 0x0E28),    /* REG_IQK_CTRL */
    };

    uint32_t rf_backup_a = rtw_rf_read(rtwdev, RF_PATH_A, 0x00);
    uint32_t rf_backup_b = rtw_rf_read(rtwdev, RF_PATH_B, 0x00);

    /* ===== PASO 2: Configurar para IQK ===== */

    /* Deshabilitar transmisión durante calibración */
    rtw_write8(rtwdev, 0x02F4, 0xFF);    /* TX Pause */

    /* Path A: IQK mode */
    rtw_write32(rtwdev, 0x0E28, 0x00000000);
    rtw_write32_mask(rtwdev, 0x0E28, 0x000000FF, 0x08);

    /* ===== TX IQK - Path A ===== */

    /* Configurar TX tone */
    rtw_write32(rtwdev, REG_IQK_TONE_A, 0x18008C1C);   /* 0x0E70 */
    rtw_write32(rtwdev, REG_IQK_TONE_B, 0x38008C1C);   /* 0x0E74 */
    rtw_write32(rtwdev, REG_IQK_AGC_PTS, 0x82110000);  /* 0x0E60 */
    rtw_write32(rtwdev, REG_IQK_AGC_RSP, 0x68160960);  /* 0x0E64 */

    /* Set TX IQK mode in RF */
    rtw_rf_write(rtwdev, RF_PATH_A, RF_IQK_REG, 0x00A00);

    /* Trigger TX IQK */
    rtw_write32(rtwdev, REG_IQK_CTRL, 0x00000001);     /* 0x0E28 */

    /* Esperar IQK completo */
    ret = rtw_poll_reg(rtwdev, REG_IQK_STATUS, BIT_IQK_DONE,
                       BIT_IQK_DONE, 10000);
    if (ret) {
        pr_err("TX IQK Path A timeout\n");
        goto restore;
    }

    /* Verificar resultado */
    val = rtw_read32(rtwdev, REG_IQK_STATUS);
    if (val & BIT_IQK_TX_FAIL) {
        pr_warn("TX IQK Path A failed\n");
    } else {
        /* Leer resultados y aplicar corrección */
        uint32_t tx_x = rtw_read32(rtwdev, REG_IQK_TX_X) & 0x3FF;
        uint32_t tx_y = rtw_read32(rtwdev, REG_IQK_TX_Y) & 0x3FF;
        pr_info("TX IQK Path A: X=0x%03X, Y=0x%03X\n", tx_x, tx_y);

        /* Aplicar corrección TX */
        rtw_write32_mask(rtwdev, 0x0C80, 0x3FF, tx_x);
        rtw_write32_mask(rtwdev, 0x0C94, 0x3FF, tx_y);
    }

    /* ===== RX IQK - Path A ===== */

    /* Configurar RX tone */
    rtw_write32(rtwdev, REG_IQK_TONE_A, 0x28008C1C);
    rtw_write32(rtwdev, REG_IQK_TONE_B, 0x48008C1C);

    /* Set RX IQK mode */
    rtw_rf_write(rtwdev, RF_PATH_A, RF_IQK_REG, 0x00C00);

    /* Trigger RX IQK */
    rtw_write32(rtwdev, REG_IQK_CTRL, 0x00000001);

    ret = rtw_poll_reg(rtwdev, REG_IQK_STATUS, BIT_IQK_DONE,
                       BIT_IQK_DONE, 10000);
    if (ret) {
        pr_err("RX IQK Path A timeout\n");
        goto restore;
    }

    val = rtw_read32(rtwdev, REG_IQK_STATUS);
    if (val & BIT_IQK_RX_FAIL) {
        pr_warn("RX IQK Path A failed\n");
    } else {
        uint32_t rx_x = rtw_read32(rtwdev, REG_IQK_RX_X) & 0x3FF;
        uint32_t rx_y = rtw_read32(rtwdev, REG_IQK_RX_Y) & 0x3FF;
        pr_info("RX IQK Path A: X=0x%03X, Y=0x%03X\n", rx_x, rx_y);

        /* Aplicar corrección RX */
        rtw_write32_mask(rtwdev, 0x0C14, 0x3FF, rx_x);
        rtw_write32_mask(rtwdev, 0x0C1C, 0x3FF, rx_y);
    }

    /* ===== Repetir para Path B ===== */
    /* (misma lógica con registros de Path B) */
    /* Path B TX IQK: RF_PATH_B, 0x0E78, 0x0E7C, 0x0E80... */
    /* Path B RX IQK: similar... */

restore:
    /* ===== PASO 3: Restaurar registros ===== */
    rtw_write32(rtwdev, 0x0800, bb_backup[0]);
    rtw_write32(rtwdev, 0x0900, bb_backup[1]);
    rtw_write32(rtwdev, 0x0C00, bb_backup[2]);
    rtw_write32(rtwdev, 0x0E28, bb_backup[3]);
    rtw_rf_write(rtwdev, RF_PATH_A, 0x00, rf_backup_a);
    rtw_rf_write(rtwdev, RF_PATH_B, 0x00, rf_backup_b);

    /* Reanudar transmisión */
    rtw_write8(rtwdev, 0x02F4, 0x00);

    return ret;
}
```

### 13.2 Acceso Indirecto a Registros RF

```c
#define RF_PATH_A    0
#define RF_PATH_B    1

/* Registros de acceso LSSI (Low-Speed Serial Interface) */
#define REG_LSSI_WRITE_A     0x0838   /* Path A write */
#define REG_LSSI_WRITE_B     0x083C   /* Path B write */
#define REG_LSSI_READ_A      0x08A0   /* Path A read */
#define REG_LSSI_READ_B      0x08A4   /* Path B read */

/* PI (Parallel Interface) Mode Registers */
#define REG_HSSI_PARM2_A     0x0828
#define REG_HSSI_PARM2_B     0x0830
#define BIT_READ_ADDR         GENMASK(27,20)
#define BIT_READ_DATA         GENMASK(19,0)
#define BIT_PI_READ_EN        BIT(8)

/* Trigger register for SI mode */
#define REG_SI_READ_A         0x08B8
#define REG_SI_READ_B         0x08BC

void rtw_rf_write(struct rtw_dev *rtwdev, uint8_t path,
                   uint8_t addr, uint32_t data)
{
    uint32_t reg;
    uint32_t val;

    /* Seleccionar registro LSSI de escritura */
    reg = (path == RF_PATH_A) ? REG_LSSI_WRITE_A : REG_LSSI_WRITE_B;

    /* Formato: [31:20] = address, [19:0] = data */
    val = ((uint32_t)(addr & 0xFF) << 20) | (data & 0xFFFFF);

    rtw_write32(rtwdev, reg, val);

    /* Delay para que la escritura se propague al RF */
    udelay(10);
}

uint32_t rtw_rf_read(struct rtw_dev *rtwdev, uint8_t path, uint8_t addr)
{
    uint32_t hssi_reg, read_reg;
    uint32_t val;

    if (path == RF_PATH_A) {
        hssi_reg = REG_HSSI_PARM2_A;    /* 0x0828 */
        read_reg = REG_LSSI_READ_A;     /* 0x08A0 */
    } else {
        hssi_reg = REG_HSSI_PARM2_B;    /* 0x0830 */
        read_reg = REG_LSSI_READ_B;     /* 0x08A4 */
    }

    /* PASO 1: Escribir dirección RF a leer */
    val = rtw_read32(rtwdev, hssi_reg);
    val &= ~BIT_READ_ADDR;                        /* Limpiar addr field */
    val |= ((uint32_t)(addr & 0xFF) << 20);       /* Set address */
    rtw_write32(rtwdev, hssi_reg, val);

    /* PASO 2: Trigger lectura */
    /* Para 8822CE en modo SI: trigger via LSSI */
    uint32_t trigger_reg = (path == RF_PATH_A) ? REG_SI_READ_A : REG_SI_READ_B;
    rtw_write32(rtwdev, trigger_reg, 0x00000000);

    /* PASO 3: Esperar y leer resultado */
    udelay(20);  /* Esperar resultado */

    val = rtw_read32(rtwdev, read_reg);
    return val & 0xFFFFF;  /* Datos en bits [19:0] */
}
```

### 13.3 Compensación Térmica (Thermal Tracking)

```c
#define RF_T_METER          0xDE     /* RF register: thermal meter */
#define REG_THERMAL_METER   0x042    /* BB thermal read register */

/* Estructura para tracking térmico */
struct rtw_thermal_info {
    uint8_t thermal_ref;        /* Valor de referencia de EFUSE */
    uint8_t thermal_current;    /* Valor actual */
    int8_t  delta;              /* Diferencia (current - ref) */
    uint8_t pwr_trim[2];       /* TX power trim por path */
};

/* Leer temperatura actual del chip */
uint8_t rtw_get_thermal(struct rtw_dev *rtwdev)
{
    uint32_t val;

    /* Trigger medición térmica */
    rtw_rf_write(rtwdev, RF_PATH_A, RF_T_METER, 0x00060);

    /* Esperar medición (típicamente 1ms) */
    msleep(1);

    /* Leer resultado */
    val = rtw_rf_read(rtwdev, RF_PATH_A, RF_T_METER);
    return (uint8_t)((val >> 2) & 0x3F);  /* Thermal value en bits [7:2] */
}

/* Compensación térmica de TX power */
void rtw_thermal_compensation(struct rtw_dev *rtwdev)
{
    struct rtw_thermal_info *therm = &rtwdev->thermal;
    uint8_t current_thermal;
    int8_t delta;
    int8_t pwr_offset;

    /* Leer valor actual */
    current_thermal = rtw_get_thermal(rtwdev);
    therm->thermal_current = current_thermal;

    /* Calcular delta respecto a referencia de fábrica */
    delta = (int8_t)current_thermal - (int8_t)therm->thermal_ref;
    therm->delta = delta;

    /*
     * Compensar: cada grado de delta ~ 0.5 dB
     * El índice de power se ajusta en unidades de 0.5 dB
     * Fórmula: pwr_offset = -delta (reducir power si temp sube)
     */
    pwr_offset = -delta;

    /* Limitar offset */
    if (pwr_offset > 6) pwr_offset = 6;
    if (pwr_offset < -6) pwr_offset = -6;

    /* Aplicar offset a Path A y Path B */
    for (int path = 0; path < 2; path++) {
        /* Escribir offset de power en BB */
        uint32_t reg = (path == RF_PATH_A) ? 0x0C54 : 0x0C5C;

        rtw_write32_mask(rtwdev, reg, GENMASK(7,0),
                         (uint32_t)(pwr_offset & 0xFF));
    }

    pr_debug("Thermal: ref=%d, cur=%d, delta=%d, pwr_offset=%d\n",
             therm->thermal_ref, current_thermal, delta, pwr_offset);
}
```

```c
int rtw8822ce_set_channel(struct rtw_dev *rtwdev, uint8_t channel,
                            uint8_t bandwidth)
{
    const struct rtw_channel_info *ch_info = NULL;

    /* Buscar canal en la tabla */
    for (int i = 0; i < ARRAY_SIZE(channel_table); i++) {
        if (channel_table[i].channel == channel) {
            ch_info = &channel_table[i];
            break;
        }
    }
    if (!ch_info)
        return -EINVAL;

    /* ===== PASO 1: Configurar RF synthesizer ===== */
    for (int path = 0; path < 2; path++) {
        rtw_rf_write(rtwdev, path, 0x18, ch_info->rf_reg_val);
    }

    /* ===== PASO 2: Configurar banda BB ===== */
    if (ch_info->band == 0) {
        /* 2.4 GHz */
        rtw_write32_mask(rtwdev, 0x0800, BIT(24), 1);    /* CCK enable */
        rtw_write32_mask(rtwdev, 0x0800, BIT(25), 1);    /* OFDM enable */
        rtw_write32_mask(rtwdev, 0x0900, GENMASK(3,0), 0x3); /* 2G mode */
    } else {
        /* 5 GHz */
        rtw_write32_mask(rtwdev, 0x0800, BIT(24), 0);    /* CCK disable */
        rtw_write32_mask(rtwdev, 0x0800, BIT(25), 1);    /* OFDM enable */
        rtw_write32_mask(rtwdev, 0x0900, GENMASK(3,0), 0x1); /* 5G mode */
    }

    /* ===== PASO 3: Configurar ancho de banda ===== */
    uint8_t bw_val;
    switch (bandwidth) {
    case 0: /* 20 MHz */
        bw_val = 0x00;
        rtw_write32_mask(rtwdev, 0x0800, GENMASK(1,0), 0x03);
        break;
    case 1: /* 40 MHz */
        bw_val = 0x01;
        rtw_write32_mask(rtwdev, 0x0800, GENMASK(1,0), 0x01);
        break;
    case 2: /* 80 MHz */
        bw_val = 0x02;
        rtw_write32_mask(rtwdev, 0x0800, GENMASK(1,0), 0x00);
        break;
    default:
        return -EINVAL;
    }

    /* Configurar BW en RF */
    for (int path = 0; path < 2; path++) {
        uint32_t rf_val = rtw_rf_read(rtwdev, path, 0x18);
        rf_val &= ~GENMASK(12,10);
        rf_val |= ((uint32_t)bw_val << 10);
        rtw_rf_write(rtwdev, path, 0x18, rf_val);
    }

    /* ===== PASO 4: AGC Table Select ===== */
    if (ch_info->band == 1) {
        /* 5 GHz: seleccionar tabla AGC adecuada */
        if (channel <= 64)
            rtw_write32_mask(rtwdev, 0x0958, GENMASK(7,4), 0x01);
        else if (channel <= 144)
            rtw_write32_mask(rtwdev, 0x0958, GENMASK(7,4), 0x02);
        else
            rtw_write32_mask(rtwdev, 0x0958, GENMASK(7,4), 0x03);
    } else {
        rtw_write32_mask(rtwdev, 0x0958, GENMASK(7,4), 0x00);
    }

    /* ===== PASO 5: Actualizar SIFS timing ===== */
    if (ch_info->band == 0) {
        /* 2.4 GHz SIFS */
        rtw_write16(rtwdev, 0x0516, 0x0A0A);  /* 10μs CCK, 10μs OFDM */
    } else {
        /* 5 GHz SIFS */
        rtw_write16(rtwdev, 0x0516, 0x100A);  /* 16μs OFDM */
    }

    /* ===== PASO 6: Recalibrar si cambio de banda ===== */
    if (rtwdev->current_band != ch_info->band) {
        rtw8822ce_iqk(rtwdev);
    }

    rtwdev->current_channel = channel;
    rtwdev->current_band = ch_info->band;
    rtwdev->current_bw = bandwidth;

    return 0;
}
```

/*
 * rtw8822ce_driver.c — RTL8822CE PCIe WiFi Driver
 *
 * Basado en la arquitectura rtw88 del kernel Linux
 */

#include <linux/module.h>
#include <linux/pci.h>
#include <linux/firmware.h>
#include <linux/interrupt.h>
#include <linux/dma-mapping.h>
#include <net/mac80211.h>

/* ═══════════════════════════════════════════════════ */
/*                     CONSTANTES                      */
/* ═══════════════════════════════════════════════════ */

#define DRV_NAME           "rtw8822ce"
#define DRV_VERSION        "1.0.0"
#define FW_FILE            "rtw88/rtw8822c_fw.bin"

#define PCI_VENDOR_REALTEK   0x10EC
#define PCI_DEVICE_8822CE    0xC822
#define PCI_DEVICE_8822CE_2  0xC82F

#define MMIO_SIZE            0x10000   /* 64KB */
#define TX_RING_SIZE         256
#define RX_RING_SIZE         512
#define RX_BUF_SIZE          16384

/* ═══════════════════════════════════════════════════ */
/*                   PCI ID TABLE                      */
/* ═══════════════════════════════════════════════════ */

static const struct pci_device_id rtw8822ce_id_table[] = {
    { PCI_DEVICE(PCI_VENDOR_REALTEK, PCI_DEVICE_8822CE) },
    { PCI_DEVICE(PCI_VENDOR_REALTEK, PCI_DEVICE_8822CE_2) },
    { /* sentinel */ }
};
MODULE_DEVICE_TABLE(pci, rtw8822ce_id_table);

/* ═══════════════════════════════════════════════════ */
/*                 INTERRUPT HANDLER                   */
/* ═══════════════════════════════════════════════════ */

static irqreturn_t rtw8822ce_isr(int irq, void *dev_id)
{
    struct rtw_dev *rtwdev = dev_id;
    uint32_t isr0, isr1;

    /* Leer estado de interrupciones */
    isr0 = rtw_read32(rtwdev, 0x0304);  /* REG_HISR0 */
    isr1 = rtw_read32(rtwdev, 0x030C);  /* REG_HISR1 */

    if (isr0 == 0 && isr1 == 0)
        return IRQ_NONE;  /* No es nuestra interrupción */

    /* Deshabilitar interrupciones mientras procesamos */
    rtw_write32(rtwdev, 0x0300, 0x00000000);  /* REG_HIMR0 */
    rtw_write32(rtwdev, 0x0308, 0x00000000);  /* REG_HIMR1 */

    /* Limpiar flags de interrupción (write-1-to-clear) */
    rtw_write32(rtwdev, 0x0304, isr0);
    rtw_write32(rtwdev, 0x030C, isr1);

    /* ---- Procesar interrupciones ---- */

    /* RX OK */
    if (isr0 & BIT(0)) {
        rtw_pci_rx_handler(rtwdev);
    }

    /* RX Descriptor Unavailable */
    if (isr0 & BIT(1)) {
        pr_warn("RTL8822CE: RX descriptor unavailable\n");
        /* Reinicializar RX ring si es necesario */
    }

    /* TX Done por queue */
    if (isr0 & BIT(2))  /* VO Done */
        rtw_pci_tx_done(rtwdev, RTW_TX_QUEUE_VO);
    if (isr0 & BIT(3))  /* VI Done */
        rtw_pci_tx_done(rtwdev, RTW_TX_QUEUE_VI);
    if (isr0 & BIT(4))  /* BE Done */
        rtw_pci_tx_done(rtwdev, RTW_TX_QUEUE_BE);
    if (isr0 & BIT(5))  /* BK Done */
        rtw_pci_tx_done(rtwdev, RTW_TX_QUEUE_BK);
    if (isr0 & BIT(6))  /* MGT Done */
        rtw_pci_tx_done(rtwdev, RTW_TX_QUEUE_MGT);
    if (isr0 & BIT(7))  /* HIGH Done */
        rtw_pci_tx_done(rtwdev, RTW_TX_QUEUE_HI);

    /* Beacon */
    if (isr0 & BIT(13))  /* BCN Interrupt */
        rtw_pci_bcn_handler(rtwdev);
    if (isr0 & BIT(25))  /* TX Beacon OK */
        pr_debug("Beacon TX OK\n");

    /* C2H (Firmware event) */
    if (isr0 & BIT(11)) {
        /* C2H se recibe como paquete en RX ring
         * ya fue procesado en rtw_pci_rx_handler */
    }

    /* FIFO overflow */
    if (isr1 & BIT(8))
        pr_warn("RTL8822CE: RX FIFO overflow\n");
    if (isr1 & BIT(9))
        pr_warn("RTL8822CE: TX FIFO overflow\n");

    /* Re-habilitar interrupciones */
    rtw_write32(rtwdev, 0x0300, rtwdev->imr0);  /* REG_HIMR0 */
    rtw_write32(rtwdev, 0x0308, rtwdev->imr1);  /* REG_HIMR1 */

    return IRQ_HANDLED;
}

/* TX Done handler — liberar recursos */
void rtw_pci_tx_done(struct rtw_dev *rtwdev, int queue_id)
{
    struct rtw_tx_ring *ring = &rtwdev->tx_ring[queue_id];

    /* Leer HW read pointer desde registro */
    uint32_t idx_reg;
    switch (queue_id) {
    case RTW_TX_QUEUE_VO:  idx_reg = 0x03B8; break;
    case RTW_TX_QUEUE_VI:  idx_reg = 0x03BC; break;
    case RTW_TX_QUEUE_BE:  idx_reg = 0x03C0; break;
    case RTW_TX_QUEUE_BK:  idx_reg = 0x03C4; break;
    case RTW_TX_QUEUE_MGT: idx_reg = 0x03C8; break;
    case RTW_TX_QUEUE_HI:  idx_reg = 0x03D0; break;
    default: return;
    }

    uint32_t hw_idx = rtw_read32(rtwdev, idx_reg) >> 16;  /* HW idx in upper 16 */

    /* Liberar SKBs entre rd_idx y hw_idx */
    while (ring->rd_idx != hw_idx) {
        struct sk_buff *skb = ring->skbs[ring->rd_idx];
        if (skb) {
            /* Unmap DMA */
            dma_unmap_single(rtwdev->dev,
                             ring->bd[ring->rd_idx].phy_addr_low,
                             skb->len, DMA_TO_DEVICE);
            /* Reportar a mac80211 */
            ieee80211_tx_status_irqsafe(rtwdev->hw, skb);
            ring->skbs[ring->rd_idx] = NULL;
        }
        ring->rd_idx = (ring->rd_idx + 1) % ring->bd_count;
    }
}

/* ═══════════════════════════════════════════════════ */
/*              mac80211 OPERATIONS                    */
/* ═══════════════════════════════════════════════════ */

static void rtw8822ce_mac80211_tx(struct ieee80211_hw *hw,
                                   struct ieee80211_tx_control *control,
                                   struct sk_buff *skb)
{
    struct rtw_dev *rtwdev = hw->priv;
    struct ieee80211_tx_info *tx_info = IEEE80211_SKB_CB(skb);
    struct ieee80211_hdr *hdr = (struct ieee80211_hdr *)skb->data;
    int queue_id;

    /* Determinar queue */
    if (ieee80211_is_mgmt(hdr->frame_control))
        queue_id = RTW_TX_QUEUE_MGT;
    else
        queue_id = rtw_tid_to_queue[skb->priority & 0x7];

    /* Preparar TX descriptor */
    /* (Prepend TX desc de 40 bytes antes del frame) */
    uint8_t *txdesc = skb_push(skb, 40);
    memset(txdesc, 0, 40);

    /* Llenar TX descriptor */
    uint32_t *dw = (uint32_t *)txdesc;

    /* DWORD0: PKT_SIZE, OFFSET, QUEUE_SEL */
    uint16_t pkt_size = skb->len - 40;  /* Sin contar descriptor */
    dw[0]  = (pkt_size & 0xFFFF) << 16;
    dw[0] |= (10 & 0x1F) << 14;        /* OFFSET = 40/4 = 10 */
    dw[0] |= (queue_id & 0x1F) << 5;

    /* DWORD1: RATE, AGG, SEC */
    if (ieee80211_is_data(hdr->frame_control)) {
        dw[1] |= BIT(7);   /* AGG_EN */
        dw[1] |= RTW_RATE_MCS7;  /* Default rate (será overridden por RA) */
    } else {
        dw[1] |= RTW_RATE_OFDM6; /* Management: OFDM 6M */
        dw[1] |= BIT(0);          /* USE_RATE (fixed rate) */
    }

    /* DWORD2: RTS/CTS, retry */
    dw[2] |= (6 & 0x3F) << 24;   /* RETRY_LMT = 6 */
    dw[2] |= BIT(23);              /* RETRY_LMT_EN */

    /* Enviar al hardware */
    int ret = rtw_pci_tx(rtwdev, skb, queue_id);
    if (ret) {
        ieee80211_free_txskb(hw, skb);
    }
}

static int rtw8822ce_mac80211_start(struct ieee80211_hw *hw)
{
    struct rtw_dev *rtwdev = hw->priv;
    int ret;

    mutex_lock(&rtwdev->mutex);

    /* Power on */
    ret = rtw8822ce_power_on(rtwdev);
    if (ret)
        goto out;

    /* Descargar firmware */
    ret = rtw8822ce_download_firmware(rtwdev, rtwdev->fw_data, rtwdev->fw_size);
    if (ret)
        goto out;

    /* Inicializar MAC */
    rtwdev->chip->ops->mac_init(rtwdev);

    /* Inicializar BB */
    rtwdev->chip->ops->bb_init(rtwdev);

    /* Inicializar RF */
    rtwdev->chip->ops->rf_init(rtwdev);

    /* Configurar canal inicial (canal 1, 20 MHz) */
    rtwdev->chip->ops->set_channel(rtwdev, 1, 0);

    /* IQK inicial */
    rtwdev->chip->ops->iqk(rtwdev);

    /* Configurar coexistencia BT */
    rtwdev->chip->ops->coex_set_init(rtwdev);

    /* Habilitar interrupciones */
    rtwdev->imr0 = BIT(0)  |  /* RX OK */
                   BIT(1)  |  /* RDU */
                   BIT(2)  |  /* VO Done */
                   BIT(3)  |  /* VI Done */
                   BIT(4)  |  /* BE Done */
                   BIT(5)  |  /* BK Done */
                   BIT(6)  |  /* MGT Done */
                   BIT(7)  |  /* HIGH Done */
                   BIT(11) |  /* C2H */
                   BIT(13) |  /* BCN Int */
                   BIT(25);   /* TX BCN OK */

    rtwdev->imr1 = BIT(8) | BIT(9);  /* FIFO overflow */

    rtw_write32(rtwdev, 0x0300, rtwdev->imr0);
    rtw_write32(rtwdev, 0x0308, rtwdev->imr1);

    /* Iniciar work de watchdog */
    schedule_delayed_work(&rtwdev->watch_dog_work, msecs_to_jiffies(2000));

    ret = 0;

out:
    mutex_unlock(&rtwdev->mutex);
    return ret;
}

static void rtw8822ce_mac80211_stop(struct ieee80211_hw *hw)
{
    struct rtw_dev *rtwdev = hw->priv;

    mutex_lock(&rtwdev->mutex);

    cancel_delayed_work_sync(&rtwdev->watch_dog_work);
    cancel_delayed_work_sync(&rtwdev->thermal_work);

    /* Deshabilitar interrupciones */
    rtw_write32(rtwdev, 0x0300, 0x00000000);
    rtw_write32(rtwdev, 0x0308, 0x00000000);

    /* Power off */
    rtw8822ce_power_off(rtwdev);

    mutex_unlock(&rtwdev->mutex);
}

static int rtw8822ce_mac80211_add_interface(struct ieee80211_hw *hw,
                                             struct ieee80211_vif *vif)
{
    struct rtw_dev *rtwdev = hw->priv;

    mutex_lock(&rtwdev->mutex);

    rtwdev->vif = vif;

    /* Configurar MAC address del hardware */
    rtw_write32(rtwdev, 0x0460,
                get_unaligned_le32(&vif->addr[0]));   /* REG_MACID Low */
    rtw_write16(rtwdev, 0x0464,
                get_unaligned_le16(&vif->addr[4]));   /* REG_MACID High */

    /* Configurar RCR según tipo de interfaz */
    uint32_t rcr = rtw_read32(rtwdev, 0x0600);        /* REG_RCR */

    switch (vif->type) {
    case NL80211_IFTYPE_STATION:
        rcr |= BIT(1) | BIT(2) | BIT(3);  /* APM | AM | AB */
        rcr |= BIT(6) | BIT(7);            /* CBSSID_DATA | CBSSID_BCN */
        break;
    case NL80211_IFTYPE_AP:
        rcr |= BIT(1) | BIT(2) | BIT(3);
        rcr &= ~(BIT(6) | BIT(7));
        break;
    case NL80211_IFTYPE_MONITOR:
        rcr |= BIT(0);  /* AAP - Accept All Packets */
        rcr |= BIT(8);  /* Accept CRC errors */
        break;
    default:
        break;
    }

    rtw_write32(rtwdev, 0x0600, rcr);

    mutex_unlock(&rtwdev->mutex);
    return 0;
}

static void rtw8822ce_mac80211_remove_interface(struct ieee80211_hw *hw,
                                                 struct ieee80211_vif *vif)
{
    struct rtw_dev *rtwdev = hw->priv;

    mutex_lock(&rtwdev->mutex);
    rtwdev->vif = NULL;
    mutex_unlock(&rtwdev->mutex);
}

static int rtw8822ce_mac80211_config(struct ieee80211_hw *hw, u32 changed)
{
    struct rtw_dev *rtwdev = hw->priv;
    struct ieee80211_conf *conf = &hw->conf;

    mutex_lock(&rtwdev->mutex);

    if (changed & IEEE80211_CONF_CHANGE_CHANNEL) {
        uint8_t ch = conf->chandef.chan->hw_value;
        uint8_t bw = 0;  /* Default 20 MHz */

        switch (conf->chandef.width) {
        case NL80211_CHAN_WIDTH_20:
        case NL80211_CHAN_WIDTH_20_NOHT:
            bw = 0;
            break;
        case NL80211_CHAN_WIDTH_40:
            bw = 1;
            break;
        case NL80211_CHAN_WIDTH_80:
            bw = 2;
            break;
        default:
            bw = 0;
            break;
        }

        rtwdev->chip->ops->set_channel(rtwdev, ch, bw);
        rtwdev->chip->ops->set_tx_power(rtwdev, ch);
    }

    if (changed & IEEE80211_CONF_CHANGE_PS) {
        if (conf->flags & IEEE80211_CONF_PS)
            rtw_pci_enter_lps(rtwdev, PS_MODE_DTIM);
        else
            rtw_pci_leave_lps(rtwdev);
    }

    mutex_unlock(&rtwdev->mutex);
    return 0;
}

static void rtw8822ce_mac80211_bss_info_changed(struct ieee80211_hw *hw,
                                                  struct ieee80211_vif *vif,
                                                  struct ieee80211_bss_conf *bss,
                                                  u64 changed)
{
    struct rtw_dev *rtwdev = hw->priv;

    mutex_lock(&rtwdev->mutex);

    if (changed & BSS_CHANGED_ASSOC) {
        if (vif->cfg.assoc) {
            /* Asociado: configurar BSSID */
            rtw_write32(rtwdev, 0x0468,
                        get_unaligned_le32(&bss->bssid[0]));
            rtw_write16(rtwdev, 0x046C,
                        get_unaligned_le16(&bss->bssid[4]));

            /* Habilitar TSF update */
            uint8_t bcn_ctrl = rtw_read8(rtwdev, 0x055A);
            bcn_ctrl &= ~BIT(4);  /* Enable TSF update */
            rtw_write8(rtwdev, 0x055A, bcn_ctrl);

            /* Configurar Aid */
            /* Notificar firmware: media status report */
            uint8_t h2c[8] = {0};
            h2c[0] = 0x01;  /* H2C_CMD_MEDIA_STATUS_RPT */
            h2c[1] = 0x01;  /* Connected */
            h2c[2] = 0x00;  /* MACID 0 */
            rtw_fw_send_h2c_command(rtwdev, h2c);

        } else {
            /* Desasociado */
            uint8_t h2c[8] = {0};
            h2c[0] = 0x01;
            h2c[1] = 0x00;  /* Disconnected */
            rtw_fw_send_h2c_command(rtwdev, h2c);
        }
    }

    if (changed & BSS_CHANGED_BEACON_INT) {
        rtw_write16(rtwdev, 0x0564, bss->beacon_int);
    }

    if (changed & BSS_CHANGED_ERP_SLOT) {
        /* Short slot: 9μs, Long slot: 20μs */
        rtw_write8(rtwdev, 0x0520, bss->use_short_slot ? 9 : 20);
    }

    if (changed & BSS_CHANGED_ERP_PREAMBLE) {
        /* Short/Long preamble setting */
        uint32_t rrsr = rtw_read32(rtwdev, 0x0228);
        if (bss->use_short_preamble)
            rrsr |= BIT(23);
        else
            rrsr &= ~BIT(23);
        rtw_write32(rtwdev, 0x0228, rrsr);
    }

    mutex_unlock(&rtwdev->mutex);
}

static void rtw8822ce_mac80211_configure_filter(struct ieee80211_hw *hw,
                                                  unsigned int changed_flags,
                                                  unsigned int *total_flags,
                                                  u64 multicast)
{
    struct rtw_dev *rtwdev = hw->priv;
    uint32_t rcr;

    *total_flags &= (FIF_ALLMULTI | FIF_OTHER_BSS | FIF_FCSFAIL |
                     FIF_BCN_PRBRESP_PROMISC | FIF_PROBE_REQ);

    mutex_lock(&rtwdev->mutex);

    rcr = rtw_read32(rtwdev, 0x0600);  /* REG_RCR */

    if (*total_flags & FIF_ALLMULTI)
        rcr |= BIT(2);   /* Accept Multicast */
    else
        rcr &= ~BIT(2);

    if (*total_flags & FIF_FCSFAIL)
        rcr |= BIT(8);   /* Accept CRC32 Error */
    else
        rcr &= ~BIT(8);

    if (*total_flags & FIF_OTHER_BSS) {
        rcr &= ~BIT(6);  /* Don't check BSSID (data) */
        rcr &= ~BIT(7);  /* Don't check BSSID (beacon) */
    } else {
        rcr |= BIT(6) | BIT(7);
    }

    rtw_write32(rtwdev, 0x0600, rcr);

    /* Configurar multicast filter */
    if (*total_flags & FIF_ALLMULTI) {
        rtw_write32(rtwdev, 0x0620, 0xFFFFFFFF);  /* MAR Low */
        rtw_write32(rtwdev, 0x0624, 0xFFFFFFFF);  /* MAR High */
    } else {
        rtw_write32(rtwdev, 0x0620, (uint32_t)(multicast & 0xFFFFFFFF));
        rtw_write32(rtwdev, 0x0624, (uint32_t)(multicast >> 32));
    }

    mutex_unlock(&rtwdev->mutex);
}

/* Security CAM operations */
int rtw_sec_add_key(struct rtw_dev *rtwdev, struct ieee80211_vif *vif,
                     struct ieee80211_sta *sta,
                     struct ieee80211_key_conf *key)
{
    uint32_t cam_val[2];
    uint8_t cam_idx;
    uint8_t sec_type;

    /* Determinar tipo de seguridad */
    switch (key->cipher) {
    case WLAN_CIPHER_SUITE_WEP40:
        sec_type = 1;
        break;
    case WLAN_CIPHER_SUITE_WEP104:
        sec_type = 5;
        break;
    case WLAN_CIPHER_SUITE_TKIP:
        sec_type = 2;
        break;
    case WLAN_CIPHER_SUITE_CCMP:
        sec_type = 4;
        break;
    default:
        return -EOPNOTSUPP;
    }

    cam_idx = key->keyidx;

    /* Escribir key en Security CAM */
    /* Cada entrada CAM tiene 8 DWORDs (32 bytes) */
    /* DWORD0: [15:0] = key info, [31:16] = MAC addr[15:0] */
    /* DWORD1: [31:0] = MAC addr[47:16] */
    /* DWORD2-7: Key material (hasta 256 bits) */

    /* Primero escribir key material */
    for (int i = 0; i < DIV_ROUND_UP(key->keylen, 4); i++) {
        uint32_t kval = 0;
        memcpy(&kval, key->key + i * 4, min(4U, key->keylen - i * 4));

        /* CAM Write: address = cam_idx * 8 + 2 + i */
        rtw_write32(rtwdev, 0x0684, kval);             /* REG_CAMWRITE */

        uint32_t cmd = BIT(31) | BIT(16);               /* Polling | Write */
        cmd |= ((cam_idx * 8 + 2 + i) & 0xFF);          /* CAM address */
        rtw_write32(rtwdev, 0x0680, cmd);                /* REG_CAMCMD */

        /* Wait for complete */
        rtw_poll_reg(rtwdev, 0x0680, BIT(31), 0, 1000);
    }

    /* Escribir entry header (DWORD0) */
    cam_val[0]  = BIT(15);                              /* Valid */
    cam_val[0] |= (sec_type & 0x07) << 2;              /* Encryption type */
    cam_val[0] |= (key->keyidx & 0x03);                /* Key index */

    if (sta) {
        cam_val[0] |= ((uint32_t)sta->addr[0] << 16) |
                       ((uint32_t)sta->addr[1] << 24);
        cam_val[1]  = ((uint32_t)sta->addr[2])       |
                       ((uint32_t)sta->addr[3] << 8)  |
                       ((uint32_t)sta->addr[4] << 16) |
                       ((uint32_t)sta->addr[5] << 24);
    } else {
        cam_val[0] |= 0xFFFF0000;
        cam_val[1]  = 0xFFFFFFFF;
    }

    /* Write DWORD1 */
    rtw_write32(rtwdev, 0x0684, cam_val[1]);
    rtw_write32(rtwdev, 0x0680, BIT(31) | BIT(16) | (cam_idx * 8 + 1));
    rtw_poll_reg(rtwdev, 0x0680, BIT(31), 0, 1000);

    /* Write DWORD0 */
    rtw_write32(rtwdev, 0x0684, cam_val[0]);
    rtw_write32(rtwdev, 0x0680, BIT(31) | BIT(16) | (cam_idx * 8 + 0));
    rtw_poll_reg(rtwdev, 0x0680, BIT(31), 0, 1000);

    /* Enable security engine */
    uint8_t sec_cfg = rtw_read8(rtwdev, 0x0690);       /* REG_SECCFG */
    sec_cfg |= BIT(0) | BIT(1) | BIT(2) | BIT(3);      /* TX/RX dec/enc */
    rtw_write8(rtwdev, 0x0690, sec_cfg);

    key->hw_key_idx = cam_idx;
    key->flags |= IEEE80211_KEY_FLAG_GENERATE_IV;

    return 0;
}
